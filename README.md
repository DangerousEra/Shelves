# 🚀 SHELVES — Next-Gen E-Commerce Experience

SHELVES is a visually intense, high-performance animated landing page designed for a modern e-commerce marketplace.
Built with advanced CSS animations, GPU-optimized effects, and bold typography to create an immersive “coming soon” experience.

---

## ⚡ Key Features

### 🎨 Crazy Visual Experience

* Glitch typography with layered color effects
* Floating geometric elements with independent animations
* Dynamic background marquee strips
* Micro flicker overlay for cinematic feel
* Noise + scanline overlays for texture

---

### 🚀 Performance-Focused

* GPU-optimized animations (no layout thrashing)
* Transform-based motion (translate, rotate only)
* Minimal repaint strategy
* Smooth mobile responsiveness

---

### 📱 Fully Responsive

* Mobile-first fixes for forms and layout
* Adaptive typography using clamp()
* Clean stacking for smaller screens

---

### 🧩 Pages Included

#### 🏠 Landing Page

* High-impact hero section
* Email capture form
* Animated UI components
* Feature grid & stats section
* Marketplace branding visuals

#### 🔐 Privacy Policy Page

* Structured legal layout
* Clean navigation + TOC
* Fully styled sections
* User data transparency-focused design

---

## 🛠 Tech Stack

* HTML5
* Advanced CSS (Animations, Keyframes, Variables)
* No frameworks (pure performance-first build)
